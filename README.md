## 📖 About This Specialization

The *Python for Everybody* specialization is a 5-course program designed to teach programming fundamentals through Python. Starting from zero prior experience, the courses progressively build skills from basic syntax all the way to data retrieval, databases, and data visualization.

This repository contains all assignments, notebooks, and supporting files for every course in the specialization.

## 📚 Course Details

### 📌 Course 1 — Programming for Everybody
> Folder: [`Course1_Programming4E`](./Course1_Programming4E)

Introduction to Python programming from scratch. Covers the essential building blocks: variables, user input, conditionals, functions, and loops.

**Key assignments:** Pay calculator, grade classifier, overtime function, max/min finder with error handling.

---

### 📌 Course 2 — Python Data Structures
> Folder: [`Course2_DataStructures`](./Course2_DataStructures)

Deep dive into Python's core data structures. Learn how to store, organize, and manipulate collections of data efficiently.

**Key assignments:** String manipulation, list operations, dictionary word counting, tuple sorting.

---

### 📌 Course 3 — Using Python to Access Web Data
> Folder: [`Course3_UsingWebData`](./Course3_UsingWebData)

Learn to retrieve and process data from the internet using HTTP requests, APIs, regular expressions, and data formats like JSON and XML.

**Key assignments:** Regex sum extractor, web scraping, JSON/XML parsing, GeoJSON mapping.

---

### 📌 Course 4 — Using Databases with Python
> Folder: [`Course4_Databases`](./Course4_Databases)

Introduction to relational databases using SQLite. Covers SQL queries, data modeling, and multi-table database design through Python.

**Key assignments:** Email database, music tracks DB, roster management, geodata visualization.

---

### 📌 Course 5 — Retrieving, Processing and Visualizing Data
> Folder: [`Course5_Visualization`](./Course5_Visualization)

Capstone course combining all previous skills. Build real applications that retrieve, process, and visualize data from the web.

**Key assignments:** PageRank algorithm, geodata mapping, data visualization pipeline.
